# Word Book 项目

## 一、项目简介

此在线词典项目使用 Go 语言及相关框架开发：使用 Gin 框架构建 RESTful API；使用 GORM 框架进行数据库操作，数据库为MySql；使用 Redis 进行缓存层的构建；JWT 进行用户鉴权；Swagger 注释生成 API 文档；并且配备了一个简易的前端界面。

---

## 二、项目目的

此项目为 Golang 学习过程中的一次阶段性实践，巩固 Gin + GORM 框架，并在做的过程中学习了 JWT鉴权、Redis 缓存、Swagger 注释等相关技术的使用，受益良多，为后续学习和项目开发夯实了基础，最重要的是让我对 Golang 的 web 开发有了完整的实践过程，具备了一定的业务开发大局观和项目架构设计能力。

## 三、功能

| 模块 | 功能         |
|-----|--------------|
| 账号 | 注册、登录、登出，使用 JWT 鉴权 |
| 查词 | 输入单词查询多种释义，支持 Redis 缓存 |
| 生词本 | 添加、删除生词，查看生词列表，更新熟练度 |
